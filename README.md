## üöÄ Getting Started

### Prerequisites

- [Bun](https://bun.sh/) runtime installed.
- Gemini API Key.

### Backend Setup

1. `cd Backend`
2. `bun install`
3. Create `.env` with `GEMINI_API_KEY`.
4. `bun dev`

### Frontend Setup

1. `cd Frontend`
2. `bun install`
3. `bun dev`

Open [http://localhost:3000](http://localhost:3000) to start building.
